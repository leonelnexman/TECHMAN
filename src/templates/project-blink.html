Разработка "под ключ"
мобильного тренажера
для улучшения зрения
Отрасль: Здоровье
Технология: Фронтент на Java, Бэкенд PHP/yii2
Платформа: Android

Комплекс упражнений, с помощью которых глаза быстрее восстанавливаются после долгой работы за экраном. Различные методики вовлечения для детей и взрослых, чтобы заниматься было интереснее.

Один из проектов, где на старте у заказчика была только идея. Принято считать, что такого мало для ТЗ. Однако все получилось: приложение стало популярным и главное полезным для пользователей. Причем дважды полезным, потому что дети во время терапии учат таблицу умножения, а взрослые иностранные языки.

Упражнения можно выбирать по разным критериям

Предпочтения

Интенсивность нагрузки

Уровень знаний пользователя

Количество занятий

Моргнув, мы больше сосредотачиваемся и запоминаем увиденное.
